Keep Talking and Nobody Explodes Mod Digital Cipher

On the Subject of Digital Cipher

Let's digitalize everything...

This module consists of 2 big screens. The upper one is the Input Screen and the bottom one the Output screen.

In order to solve the module take the string written in the Input screen. Thats String A. Next take String A and reverse it, which will then be String B. Take the numerical value of the first letter in String A and the first letter in String B and add them up. Take the digital root of the sum and then subtract 1. Click the button which corresponds to this digit. Repeat for the whole sequence.

In order to get the numerical value of a letter follow this sequence: A=0, B=1, C=2 and so on.

If the digital root before subtracting 1 is already 0, use 0 instead.
